Crear tarea

Tip rapido

Prioriza tareas importantes antes que las urgentes.

Para crear una tarea sigue estos pasos:

  1. Escribe el nombre de la tarea
  2. Selecciona la fecha límite (ejemplo: 10/12/2024)
  3. Escribe la prioridad (baja, media o alta)
  4. Agrega una descripción opcional
